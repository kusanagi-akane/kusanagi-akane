## About Me

- 17 歲的全端開發者，熱愛寫程式，目前正在準備邁向資工 / 資管相關領域。
- 開發並維護大型多功能 Discord 機器人，主打專案「貓貓超可愛」已突破 **24 萬** 使用者。
- 目前擔任 **DCTW 審核員**、**KUSANA STUDIO 開發者**，並參與萬人社群管理。
- 主要專注於 Python 異步架構、API 整合、虛擬經濟系統、社群自動化與安全管理。
- 喜歡設計簡單、直覺、可愛但實用的工具與介面。
